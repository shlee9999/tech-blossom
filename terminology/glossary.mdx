# Terminology

## 개요

이 공간은 개발 과정에서 마주치는 **다양한 용어들을 체계적으로 정리하고 기록하는 용어 사전**입니다.

## 툴팁 지원

- 문서 내에서 노란색으로 하이라이트된 특정 용어에 마우스를 올리면 해당 용어의 간단한 설명이나 정의가 툴팁 형태로 자동 표시됩니다.
- 링크를 클릭하면, 이 공간의 해당 용어 페이지로 이동합니다.
- 예시 : <TermTooltip termId='annotation'>어노테이션</TermTooltip>

## 목적

- **용어 정리**: 개발 관련 용어들을 명확하게 정의하여 이해도 향상
- **지식 체계화**: 분야별로 용어를 분류하여 학습 효율성 증대
- **참고 자료**: 개발 중 헷갈리는 용어들을 빠르게 찾아볼 수 있는 레퍼런스 제공

:::info
각 분야의 용어들은 실제 개발 경험을 바탕으로 선별하여 정리되었습니다.
:::
